NLCI Malayalam fonts
====================

This file describes the source files included with the NLCI Malayalam fonts.
This information should be distributed along with the NLCI Malayalam fonts and any derivative works.

master.sfd            - OpenType rules
other .sfd files      - glyph outline data for each font
vayalar_unicode.xml   - mapping from legacy to Unicode as used by ttfbuilder
