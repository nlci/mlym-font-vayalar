FONTLOG
Vayalar, Kalyani, Malabar - NLCI Malayalam fonts
================================================

This file provides detailed information on the NLCI Malayalam fonts.
This information should be distributed along with the NLCI Malayalam fonts and any derivative works.

Basic Font Information
----------------------

Pan Malayalam font designed to support all the languages using the Malayalam script.
